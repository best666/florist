# 快捷指令库

## 设计原则

- **按需收拢** — 将项目高频操作用简短别名封装，减少重复输入。
- **可导入** — 新项目可直接复用已有指令模板，加速初始化。
- **不强制即时生成** — 日常调试仅标记高频问题，累计 ≥3 次后再生成指令。

## 指令格式

```
/指令名 — 简短描述
```

示例：
```
/start — 启动全栈开发环境（pnpm dev）
/check — 全项目类型 + 格式检查（pnpm typecheck && pnpm format:check）
/db-reset — 重置开发数据库
```

## 执行流程

1. **识别高频操作** — 在开发过程中标记出现 ≥3 次的命令序列或操作模式。
2. **生成别名** — 为高频操作创建简短、易记的别名指令。
3. **归入指令库** — 写入 `.claude/shortcuts.md` 或本技能文件末尾，便于团队共享。
